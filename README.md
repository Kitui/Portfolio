# Paul John Kitui Portfolio

Static personal portfolio for Paul John Kitui.

## Deploy

This site is ready for any static host:

- GitHub Pages: push this folder to a GitHub repository and enable Pages from the `main` branch.
- Netlify: deploy the folder as a static site.
- Vercel: import the repository as a static project.

The entry point is `index.html`; there is no build step.
